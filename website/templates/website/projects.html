Project 1: Instant Messaging App

My first hands-on introduction to the development process was a web-based messaging app for PSU's Elements of Software Engineering class in 2021. The goal of the project was to have a working browser-based chat system which could facilitate communication between at least two logged-in users, implemented using the MERN stack (MongoDB, Express, React, and Node). By the end of the quarter, I had a pretty rough working example of the app, with the ability to demonstrate messaging across two instances. However, along with many other features, the user functionality was never implemented, despite there being a user creation form.

GitHub repo: https://github.com/awong-pdx/CS300/tree/main/instant-messaging-app

Project 3: React Weather Dashboard

App link: https://awong-pdx.github.io/frontend-weather-dashboard

The weather dashboard was a final project for Front End Web Technologies, where my partner and I were tasked with creating a dashboard application using React. This was my first major experience with React, and I loved learning the intricacies and features which make React such a popular framework, as well as comparing my previous experience with Django.

For a six-week project, I was really happy with the way the application turned out! I learned so much in such a short amount of time, and improved on several skills, including Git/GitHub collaboration, wireframing, React, UI design, and overall front-end development.

GitHub repo: https://github.com/awong-pdx/frontend-weather-dashboard
